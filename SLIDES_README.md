# Slidev Presentation

This directory contains a Slidev presentation for the Claude Sonnet 4.5 frontend evaluation.

## Quick Start

To run the presentation locally:

```bash
# Start the presentation server
npm run slides

# Or use npx directly
npx slidev
```

The slides will open automatically in your browser at `http://localhost:3030`

## Available Commands

- `npm run slides` - Start the dev server for the presentation
- `npm run slides:build` - Build the presentation for hosting
- `npm run slides:export` - Export slides to PDF

## Features

The presentation includes:
- **2 Main Content Slides** focused on Claude Sonnet 4.5 evaluation
- Interactive animations and transitions
- Code syntax highlighting
- Responsive design
- Dark theme
- GitHub link integration

## Slides Content

The `slides.md` file contains:
1. **Title Slide** - Introduction
2. **Slide 1** - Project Overview & Why Claude Sonnet 4.5
3. **Slide 2-10** - Evaluation Framework (5 metrics explained)
4. **Final Slides** - Results, Achievements, and Conclusion

## Customization

The slides use Slidev's default theme with customizations in the frontmatter:
- Theme: default
- Transition: slide-left
- Syntax highlighter: shiki
- Background images from Unsplash
